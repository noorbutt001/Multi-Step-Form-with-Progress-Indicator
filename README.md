# 🚀 Multi-Step Form Wizard
Note: This is a front-end website created for my Semester project.
A fully interactive multi-step form built using HTML, CSS, and JavaScript. This project guides users through a step-by-step form with validation, progress tracking, and autosave functionality.
---
## ✨ Features
- ✅ Step-by-step form navigation  
- ✅ Dynamic progress bar  
- ✅ Form validation (required fields, email format, etc.)  
- ✅ Next & Previous navigation buttons  
- ✅ Final review before submission  
- ✅ Autosave using localStorage  
- ✅ Smooth CSS transitions  
- ✅ Responsive design  
---
## 🛠️ Technologies Used
- HTML5  
- CSS3 (Animations & Transitions)  
- JavaScript (DOM Manipulation & Validation)
